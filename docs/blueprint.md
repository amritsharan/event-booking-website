# **App Name**: Gilded Events

## Core Features:

- Event Listing: Browse a variety of events with filtering and search options.
- Event Details: View comprehensive information for each event, including description, schedule, venue details, and available ticket types.
- Ticket Booking: Select desired tickets and complete the booking process.
- AI Event Recommendation: AI tool provides personalized event suggestions based on user preferences and past booking history.
- Reservation Management: View and manage past and upcoming reservations.

## Style Guidelines:

- Primary color: Dark blue (#192A56) for a sophisticated and modern look.
- Background color: Very dark desaturated blue (#1A1E23), almost black, to set off the brighter hues in the interface.
- Accent color: Gold (#D4AF37) for highlights and CTAs, providing a luxurious touch.
- Headline font: 'Playfair' serif for headlines, bringing a touch of elegance.
- Body font: 'PT Sans' sans-serif for body text; for good readability with the stylish headlines.
- Use elegant, minimalist icons related to event categories.
- Subtle transitions and animations on navigation and interactions.
